# 위험 분석

	- 아마존 AWS 서버 시스템 configuration
	- Hadoop , Spark와 같은 빅데이터 플랫폼 사용 역량
	
# 요구사항 분석

	- 각 채널 혹은 영상에 대해서 필요한 데이터를 프로그래밍을 통해 수집 할 수 있어야 합니다.
	- 수집된 데이터는 전부 데이터베이스에 저장되어야 합니다.
	- 수집된 데이터 분석을 통해 의미있는 값을 도출 할 수 있어야 합니다.
	- 도출된 값을 웹 I/O를 통해 접근성을 가질 수 있어야 합니다.
